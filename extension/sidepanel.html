    PUBLIC SETUP

    Connect to Hermes

    1. Start the Hermes API server locally or on a reachable remote machine.
    2. Clone this repo, build it, and load the dist/ folder unpacked.
    3. Open the Hermes extension side panel from your browser toolbar.
    4. Click Connect to Hermes, or use Manual setup with a local/remote Gateway URL and token.
    5. Approve pairing when available, or paste API_SERVER_KEY for manual setup.
    6. Ask Hermes about the active page context.

    Public alpha: load unpacked, local or remote Hermes API server, full Hermes tools when the connected runtime supports them.
